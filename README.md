# TodoApp / next

nextjs로 만든 Todo App입니다.

## Features

- TodoList 보여주기
- Todo 추가하기
- 실행한 Todo 체크하기
- 실행한 Todo 삭제하기
- APIserver

## I**nstallation**

```
git clone https://github.com/idbdyoung/TodoApp-next.git
cd TodoApp-next
npm install
```

## Execution

3000번 포트를 사용하고 있습니다.

**Dev Mode**

```jsx
npm run dev

//next dev
```

**Build**

```jsx
npm run build

//next build
```

**Start**

```jsx
npm start

//next start
```
